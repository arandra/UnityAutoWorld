# UI 계획

## 개요
- 코어와 연결하여 입력 처리 및 이벤트 출력 결과를 보여준다.

## UI layout 구성
### 전체 화면을 약 8:2로 나눠 출력 : 입력으로 구성한다.

### 출력
- console mesaage 나 chatting message 처럼 문장으로 보여준다.
- 스크롤 가능하다.
- 메시지의 중요도에 따라 색을 구분한다. (흰색, 초록, 파랑, 노랑, 빨강)

### 입력
- 메뉴 선택 방식으로 선택에 따라 트리형태로 진행후 마지막 노드라면 입력을 진행한다.
- 상위 메뉴로 갈 수 있다.
- 스크롤이 가능하다.
- 트리구성

1.1.1. 증가
1.1.2. 감소
